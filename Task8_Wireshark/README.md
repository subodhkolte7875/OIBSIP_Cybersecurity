# Task 8: Network Traffic Analysis using Wireshark

## Objective
To capture and analyze ICMP network traffic using Wireshark.

## Tool Used
- Wireshark
- Kali Linux (VirtualBox)

## Steps Performed
1. Started Wireshark and selected eth0 interface.
2. Generated ICMP traffic using ping command.
3. Applied ICMP display filter.
4. Captured and analyzed ICMP packets.

## Files Included
- task8_capture.pcapng
- task8_wireshark_icmp.png

## Conclusion
This task demonstrates basic packet capturing and traffic analysis using Wireshark.
